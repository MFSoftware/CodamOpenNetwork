# Codam Open Network
Opensource network for private messaging

## How to build
```bash
npm run build
```

## How to run
Execute
```bash
npm run start
```
to run in **production**, or
```bash
npm run dev
```
to run in **development** mode.

## Start test
```bash
npm run test
```
